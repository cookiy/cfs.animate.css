cfs.animate.css
===============

css3的运动样式库

利用这个css文件，我们就能很快的使用一些好玩的动画样式。

例如

```<link rel="stylesheet" type="text/css" href="CFS.Animate.min.css">```

引入我们的样式

对您的html元素添加class

例如

```<h1 class="bounceIn">CFS.ANIMATE.CSS</h1>```

那么 他就会有一个动画效果了

参考样例 ： http://animate.coolfishstudio.com/
